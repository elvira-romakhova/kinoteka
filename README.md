# кинотека  
домашняя кинотека для итоговой аттестации   

Ромахова Эльвира  

Описание:  
Простое приложение с графическим интерфейсом для хранения информации о фильмах.  
Позволяет добавлять фильмы, указывая название, жанр, год выпуска и рейтинг.  
Все данные сохраняются в JSON-файл и автоматически загружаются при запуске.  

Функциональность:  
-Добавление фильмов с валидацией:  
-Год - целое число от 1895 до текущего  
- Рейтинг - число от 0 до 10 (включая десятичные дроби, до 5 знаков после запятой)  
- Отображение списка фильмов в таблице  
Фильтрация:  
- По жанру  
- По году  
Удаление выбранного фильма  
Сохранение и загрузка через JSON (файл "data_movies.json" создаётся автоматически)  
Запуск:
1. Для запуска приложения должен быть установлен Python 3.x.
2. Скачайте файл Elvira.py.
3. Откройте терминал/командную строку в папке с файлом.
4. Введите команду:bashpython Elvira.py

Пример использования:
1. Запустите программу.  
2. Заполните поля:  
Название: "Побег из Шоушенка"  
Жанр: "Драма"  
Год: 1895  
Рейтинг: 9.3  
3. Нажмите "Добавить фильм" - фильм появится в таблице.  
4. Для фильтрации по жанру введите, например, "драма" и нажмите "Применить фильтр".  
5. Для сброса фильтрации используйте кнопку "Сбросить фильтр".  
6. Чтобы удалить фильм, выделите его в таблице и нажмите "Удалить выбранный фильм". Выделение одного работает при простом нажатии на выбранный фильм, для отмены выделения или выделения нескольких используйте ctrl
